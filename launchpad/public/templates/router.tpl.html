Single-Page Application Router

Simple Demo Application

A simple web-application with a few static pages. This is the first demo site I created, with little or no concern for advanced functionality. The pages contain primarily text-based templates. None of the routes are dynamic in any way. The "Page Not Found" route was retrofitted after it was intially added to the capstone router.

Programmatic Navigation (With 404)

A test bed created to showcase programmatic page navigation, including routing to a catchall route when a page is not found.

Heavy Bundle Simulation

A simple static site including various lightweight and heavy pages. Included to showcase the both the lazy-loading and caching of expensive navigation operations.

Route Parameter Demo

A demo using route parameters and and template factories to dynamically construct the template for the page the user has requested. Interfaces with the GitHub API to demonstrate promise-based template factories.

Synchronous Development Spike

This is the proof-of-concept project that I created initially before starting actual development. It is a SPA router that is fully synchronous, using a single bundle for all of the templates rather than loading templates lazily as they are needed.
